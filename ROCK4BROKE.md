# 🤘 穷玩摇滚

一份为预算有限的电吉他新手准备的入门指南

## ❓ 为什么会有这份指南？

互联网上从不缺吉他教学，但一个新手上路时最困惑的三个问题，却很少被系统回答：

- 电吉他到底是什么？ —— 它和民谣吉他有什么区别？它是怎么出声的？

- 我到底需要买哪些东西？ —— 除了琴，那一堆线、效果器、音箱都是干什么的？哪些非买不可，哪些可以等等？

- 究竟要花多少钱？ —— 能不能有一个从“能响”到“够玩”的清晰路线图，让我心里有底？

这份指南就是为了回答这三个“元问题”而写的。它是一份 “入圈基建”手册，旨在用最有限的预算，帮你搭建一个能正常运转、并能让你理解其所以然的电吉他系统。

我们的目标不是一步到位，而是聪明地开始。

## 🤔 电吉他 ABC

***本部分可能涉及专业的数学、物理理论，请各位读者按照自身实际接受能力阅读***

### 问：电吉他如何发声？

- 电吉他通过拾音器和音箱发声。拾音器装有磁铁和线圈。磁铁磁化附近的琴弦钢材，琴弦振动时引起线圈内磁通量发生变化从而产生感应电动势和感应电流。这一信号再经过处理与放大，输出给音箱并播放，这就是电吉他的声音。

### 问：为什么不能给电吉他直接连耳机？

- 这个问题需要用接口间的阻抗关系来解释。我们先来复习一下最大功率传输定理：

    `当负载阻抗与功率输出接口的除源阻抗共轭时（对于纯电阻电路则是相等），负载获得的功率最大。`

- 对于一个**从接口传出的信号**，若接口的除源阻抗（一般称为**输出阻抗**）较大，则**习惯上**称这个信号为**高阻抗信号**；反之则是**低阻抗信号**。传输某个信号的前后级接口应该具有共轭的除源阻抗以保证传输功率最大，这称为**阻抗匹配**，常用于驱动音箱等大功率设备。

    > `用“高阻抗”或“低阻抗”的说法去描述一个信号是不准确的，因为阻抗只能用于描述接口性质。但考虑到工程应用中阻抗匹配确实会对信号造成影响，于是我们可以将这种情况下的信号理解为含参信号，被包含的参数自然就是输出阻抗。`

- 电吉他虽然可以作为一个交流电源使用，但是其电动势极低（数十毫伏），而内阻极高（数千或数万欧姆）。耳机作为负载，阻抗一般不超过 100 Ω ，这意味着耳机获得的电压或功率几乎是 0。为此电吉他不能直接连耳机。

- 电吉他效果链的主要目的是尽可能无损**传输电压信号**，而一个连接处的上游阻抗远小于下游阻抗时，下游负载才能分得尽可能多的电压，这个要求被称为**阻抗桥接**，服务于信号幅值的最大化传输。电吉他效果链中几乎所有设备的输入阻抗都在1MΩ量级而输出阻抗都在1kΩ量级，以让自身和下游设备尽可能多分到电压。

- 在耳机直连电吉他的情况下，由于吉他输出阻抗远大于耳机输入阻抗，既不满足阻抗匹配（功率最大）也不满足阻抗桥接（电压最大），导致负载上获得的电压和功率都极低。相比之下，在正常的效果链中，设备间的连接遵循阻抗桥接原则（输出阻抗低、输入阻抗高），虽然这种连接并不满足最大功率传输条件，但每个设备都有独立电源供电，信号会在内部被放大到所需的功率水平，因此无需像射频电路那样追求严格的功率匹配。”

### 问：电吉他的音量旋钮有什么用？

- 它可以调节声音信号的**强弱**，读数越大声音越响。本质上是一个电位器，调节电吉他接口的输出电压。可以与箱头或增益单块的 GAIN 旋钮联合调节增益。

### 问：电吉他的音色旋钮有什么用？

- 它可以调节声音信号中**高频成分的含量**，读数越大声音越亮。本质上是一个可变低通滤波器，滤除输出信号的高频成分。可以与箱头的三段均衡或 TONE 旋钮联合调节音色。

### 问：如何对电吉他的旋钮读数？

- **极值定位法**：将旋钮拧到最大值或最小值，观察刻度"10"或刻度"0"的位置，那里就是"指针"。

### 问：如何调节四旋钮吉他的音量音色？

- 四个旋钮分别调节琴颈（靠近指板）和琴桥（琴弦固定处）拾音器的音量音色。

- 在单个拾音器工作时，只有对应的旋钮是正在工作的，您可以单独调节这个拾音器采集信号的音量音色。

- 而在多个拾音器同时工作时，拾音器之间是**并联**状态，这意味着一旦将某个音量旋钮置零，整个拾音器部分都会短路，电吉他就会**静音**而不会变成单拾音器工作。

    > `这也是双拾音器档位的快速静音技巧，当您的吉他出现啸叫时可以迅速将某个音量置零。`

- 同时，因为拾音器之间是并联关系，负载效应会带来独特的音色交互而非简单的线性组合，丰富的音色表现也是 Les Paul 的魅力所在，您可以多多尝试以找到最满意的音色。在双拾音器档位追求单个拾音器工作是舍近求远的，您完全可以拨动拨杆来切换到单拾音器。

### 问：对于多拾音器电吉他，不同拾音器采集的信号有什么不一样？

- 好问题。我们来复习一下琴弦振动的波动特征：琴弦的振动是一系列**驻波**的叠加，而驻波的频率有高有低，频率最低的称为**基波**，其频率称为**基频**，同时也代表了整个振动的频率；其他的驻波称为**谐波**，其频率称为**谐频**，大概是基频的整数倍，这一整倍数是谐波的**次数**。

    > `我们在这里说“大概是”整数倍，是因为弱阻尼振动只能保证各次谐波的复频率模（同时是无阻尼振动频率）是严格整数倍，而我们讨论的频率实际上是复频率的虚部（阻尼振动频率）。较小的阻尼只能保证这些阻尼振动频率是近似整数倍，而接近临界的阻尼连近似整数倍关系都无法保证。这一语境下谐波的次数应该理解为其复频率模的倍数。`

- 驻波的波形呈现出波峰和波谷位置不随时间变化的特征。波峰和波谷这种振幅最大的位置称为**波腹**，振幅始终为0的位置称为**波节**。波腹总出现在半波长的中点。

- 阻尼振动虽然不能保证谐波频率的整数倍关系，但是可以保证谐波波数（反比于波长）的整数倍关系。同时基波成分的半波长正好等于弦的振动长度（由固定型边界条件决定），这意味着弦上可能存在的所有驻波，其波长必须满足：弦长 L 等于半波长的整数倍。

- 因此，第 n 次谐波的波长 λ_n 与弦长 L 满足：L = n * (λ_n / 2)。换言之，第 n 次谐波会在弦上形成 n 个半波，从而在弦上产生 (n + 1) 个波节和 n 个波腹。

- 这些波腹的位置是严格确定的。对于第 n 次谐波，其波腹位于距离琴弦一端（如琴桥）(2k-1) * L / (2n) 的位置，其中 k = 1, 2, ..., n。也就是说，第 n 次谐波的 n 个波腹，会均匀地分布在整个弦长上。

- 由此可得一个重要结论：基波 (n=1) 只有一个波腹，它位于弦的正中央。而**琴颈拾音器**通常安装在靠近弦中央的区域（Les Paul 一般是靠近琴桥的琴弦四等分点），因此它天然地靠近基波和低次谐波（如2次、3次谐波）的波腹，能强烈地采集到这些决定音色主体和厚度的频率成分。

- 相反，高次谐波 (n 较大) 的波腹数量多且分布密集。**琴桥拾音器**所在的位置，虽然远离基波的波腹（Les Paul 一般是靠近琴桥的琴弦十六等分点），却很可能与某个高次谐波（例如第8次谐波）的波腹位置重合。这使其能优先采集到那些赋予音色明亮度、冲击力和“金属感”的高频成分。

### 问：单线圈拾音器和双线圈拾音器有什么区别？

- **单线圈拾音器**的特性是，它容易受到环境中电磁场（如灯光、电脑）的干扰从而采集到噪音。**双线圈拾音器**通过**两个线圈反向缠绕**，可以有效抵消这种噪音。如果噪音过大，检查一下是否面对干扰源，或者尝试在房间的不同位置练习。

### 问：综合效果器音箱是什么意思？

- 你知道的，电吉他的信号需要处理才能形成音色，信号的处理流程就是**效果链**。

- 处理信号一般需要多种变换，实现这一变换的设备就叫做**效果器**。

- 如果一个效果器只能实现一种变换，则称为**单块效果器**；如果一个效果器能实现多种变换（一般是电子的），则称为**综合效果器**。

- 在传统的由单块效果器组成的效果链中，串联顺序一般是这样的：

    `吉他 -> 调音表 -> 噪音门 -> 压缩器 -> 哇音踏板 -> 均衡 -> 增益 -> [调制 -> 延迟 -> 混响] -> 放大器 -> 音箱箱体`

- 现在我们来介绍**箱头**，其包含了**前级**和**后级**两部分，箱头的功能是塑造信号的音色并放大功率以驱动音箱。

- `均衡`和`增益`部分称为箱头的**前级**，用于塑造音色并初步放大至足以驱动后级。它负责处理来自吉他的微弱信号，通过调节增益（GAIN）来**决定音色的失真度和饱和度**（从清澈到 crunch 再到高增益失真）；通过均衡（EQ，通常是三段：低频、中频、高频）来**塑造音色的骨架**（比如增强低频让声音更厚实，提升高频让声音更明亮）。

- `放大器`部分称为箱头的**后级**，用于放大信号功率并改变信号的阻抗匹配特性。它接收来自前级或效果环路的信号，并将已经塑形完毕的**低功率、高阻抗信号**转换为**高功率、低阻抗的信号**，从而拥有足够的能量去驱动扬声器（喇叭）这种低阻抗负载。

- 箱头的前级和后级之间设有一个**效果环路（FX Loop）**，它是一个由 SEND 和 RETURN 两个接口组成的回路。乐手可以将`调制`、`延迟`、`混响`等效果器连接在这个回路中，使这些空间类效果工作在前级失真之后、后级放大之前，以获得更清晰的尾音。它们为干瘪的原始信号添加空间感和趣味性。混响**模拟在不同空间中的反射**，制造深度；延迟**制造回声或增加延音**，增加层次感；调制效果则**让声音流动、旋转起来**。这些效果器是可以选择安装与否的，如果选择不安装，前级的输出信号会直接输入给后级。

- 市面上大部分数字综合效果器都可以模拟`调音表 -> 噪音门 -> 压缩器 -> 哇音踏板 -> 均衡 -> 增益 -> 调制 -> 延迟 -> 混响`这一部分（具体构成和顺序可能不同）。再进一步，如果把数字综合效果器和`放大器`、`扬声器`集成在同一外壳内，就得到了通常所说的**综合效果器音箱**（更准确的术语是‘数字建模一体式音箱’）。它是一套完整的“一体化解决方案”，无需外接任何设备即可获得多种可编程音色。

### 问：我的琴桥怎么是斜的？

- 不用担心，这不是设计或制造瑕疵，而是**补偿式琴桥**设计。

- 这一设计首先补偿了**低音弦的弦高**，也就是到指板的距离。低音弦更粗且振动幅度更大，需要更大的弦高以防止打品，于是补偿式琴桥的高度随琴弦振动频率降低而逐渐升高。

- 其次它补偿了**低音弦的频率**。当手指按下低音弦时，琴弦的横向位移远大于同品位高音弦。为了在按弦时避免更大的横向位移造成音高偏移，把琴弦加长可以让“稀释”这一位移以稳定音准，于是补偿式琴桥的位置随琴弦振动频率降低而逐渐远离弦枕。
    > `根据白金汉 π 定理，任何一个由有量纲的物理量参与的物理过程中的函数关系都可以转换成某些无量纲量之间的函数关系。琴弦横向位移与有效长度的比 x/L 是一个非常重要的无量纲量，其在研究琴弦横向变形对振动特性的影响时记载了琴弦变形的几何特征。因为低音弦比较高，从而 x 更大，自然就要把 L 做大一点以保证不同琴弦的变形特征一致。`

### 问：摇把是什么？怎么用？

- 我们知道，琴桥的位置决定了琴弦的有效振动长度，从而和琴弦的张力一起决定琴弦的振动频率。如果我们让琴桥可以前后倾斜旋转并连接一根**摇把**用以控制倾斜角度（从而改变琴弦的张力和长度），那么我们就得到了电吉他的**颤音系统**。摇动摇把就可以实时调整琴弦的振动频率以产生颤音。
    > `颤音系统集成了浮动式琴桥、穿弦孔、弹簧系统等，整体看来是一块开了六个穿弦孔和一个摇把接口并装上六个弦鞍，铰接在琴身上并与琴身之间挂上弹簧的板子。`

- 如果一把吉他的摇把只能下压以降低音高，则称之为**单摇吉他**；如果一把吉他的摇把既能下压以降低音高又能上抬以升高音高，则称之为**双摇吉他**。

- 颤音系统的调节远比固定琴桥麻烦，尤以双摇为最。建议新手从固定式琴桥或单摇吉他开始练习。

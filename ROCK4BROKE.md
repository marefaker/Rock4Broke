## 🤔 电吉他 ABC

***本部分可能涉及专业的数学、物理理论，请各位读者按照自身实际接受能力阅读***

### 问：电吉他如何发声？

- 电吉他通过拾音器和音箱发声。拾音器装有磁铁和线圈。磁铁磁化附近的琴弦钢材，琴弦振动时引起线圈内磁通量发生变化从而产生感应电动势和感应电流。这一信号再经过处理与放大，输出给音箱并播放，这就是电吉他的声音。

### 问：为什么不能给电吉他直接连耳机？

- 这个问题需要用接口间的阻抗关系来解释。我们先来复习一下最大功率传输定理：

    `当负载阻抗与功率输出接口的除源阻抗共轭时（对于纯电阻电路则是相等），负载获得的功率最大。`

- 对于一个**从接口传出的信号**，若接口的除源阻抗（一般称为**输出阻抗**）较大，则**习惯上**称这个信号为**高阻抗信号**；反之则是**低阻抗信号**。传输某个信号的前后级接口应该具有共轭的除源阻抗以保证传输功率最大，这称为**阻抗匹配**，常用于驱动音箱等大功率设备。

    > `用“高阻抗”或“低阻抗”的说法去描述一个信号是不准确的，因为阻抗只能用于描述接口性质。但考虑到工程应用中阻抗匹配确实会对信号造成影响，于是我们可以将这种情况下的信号理解为含参信号，被包含的参数自然就是输出阻抗。`

- 电吉他虽然可以作为一个交流电源使用，但是其电动势极低（数十毫伏），而内阻极高（数千或数万欧姆）。耳机作为负载，阻抗一般不超过 100 Ω ，这意味着耳机获得的电压或功率几乎是 0。为此电吉他不能直接连耳机。

- 电吉他效果链的主要目的是尽可能无损**传输电压信号**，而一个连接处的上游阻抗远小于下游阻抗时，下游负载才能分得尽可能多的电压，这个要求被称为**阻抗桥接**，服务于信号幅值的最大化传输。电吉他效果链中几乎所有设备的输入阻抗都在 1MΩ 量级,而输出阻抗都在 1kΩ 量级，以让自身和下游设备尽可能多分到电压。

- 在耳机直连电吉他的情况下，由于吉他输出阻抗远大于耳机输入阻抗，既不满足阻抗匹配（功率最大）也不满足阻抗桥接（电压最大），导致负载上获得的电压和功率都极低。相比之下，在正常的效果链中，设备间的连接遵循阻抗桥接原则（输出阻抗低、输入阻抗高），虽然这种连接并不满足最大功率传输条件，但每个设备都有独立电源供电，信号会在内部被放大到所需的功率水平，因此无需像射频电路那样追求严格的功率匹配。

### 问：电吉他的音量旋钮有什么用？

- 它可以调节声音信号的**强弱**，读数越大声音越响。本质上是一个电位器，调节电吉他接口的输出电压。可以与箱头或增益单块的 GAIN 旋钮联合调节增益。

### 问：电吉他的音色旋钮有什么用？

- 它可以调节声音信号中**高频成分的含量**，读数越大声音越亮。本质上是一个可变低通滤波器，滤除输出信号的高频成分。可以与箱头的三段均衡或 TONE 旋钮联合调节音色。

### 问：如何对电吉他的旋钮读数？

- **极值定位法**：将旋钮拧到最大值或最小值，观察刻度"10"或刻度"0"的位置，那里就是"指针"。

### 问：如何调节四旋钮吉他的音量音色？

- 四个旋钮分别调节琴颈（靠近指板）和琴桥（琴弦固定处）拾音器的音量音色。

- 在单个拾音器工作时，只有对应的旋钮是正在工作的，您可以单独调节这个拾音器采集信号的音量音色。

- 而在多个拾音器同时工作时，拾音器之间是**并联**状态，这意味着一旦将某个音量旋钮置零，整个拾音器部分都会短路，电吉他就会**静音**而不会变成单拾音器工作。

    > `这也是双拾音器档位的快速静音技巧，当您的吉他出现啸叫时可以迅速将某个音量置零。`

- 同时，因为拾音器之间是并联关系，负载效应会带来独特的音色交互而非简单的线性组合，丰富的音色表现也是 Les Paul 的魅力所在，您可以多多尝试以找到最满意的音色。在双拾音器档位追求单个拾音器工作是舍近求远的，您完全可以拨动拨杆来切换到单拾音器。

### 问：对于多拾音器电吉他，不同拾音器采集的信号有什么不一样？

- 好问题。我们来复习一下琴弦振动的波动特征：琴弦的振动是一系列**驻波**的叠加，而驻波的频率有高有低，频率最低的称为**基波**，其频率称为**基频**，同时也代表了整个振动的频率；其他的驻波称为**谐波**，其频率称为**谐频**，大概是基频的整数倍，这一整倍数是谐波的**次数**。

    > `我们在这里说“大概是”整数倍，是因为弱阻尼振动只能保证各次谐波的复频率模（同时是无阻尼振动频率）是严格整数倍，而我们讨论的频率实际上是复频率的虚部（阻尼振动频率）。较小的阻尼只能保证这些阻尼振动频率是近似整数倍，而接近临界的阻尼连近似整数倍关系都无法保证。这一语境下谐波的次数应该理解为其复频率模的倍数。`

- 驻波的波形呈现出波峰和波谷位置不随时间变化的特征。波峰和波谷这种振幅最大的位置称为**波腹**，振幅始终为0的位置称为**波节**。波腹总出现在半波长的中点。

- 阻尼振动虽然不能保证谐波频率的整数倍关系，但是可以保证谐波波数（反比于波长）的整数倍关系。同时基波成分的半波长正好等于弦的振动长度（由固定型边界条件决定），这意味着弦上可能存在的所有驻波，其波长必须满足：弦长 L 等于半波长的整数倍。

- 因此，第 n 次谐波的波长 λ_n 与弦长 L 满足：L = n * (λ_n / 2)。换言之，第 n 次谐波会在弦上形成 n 个半波，从而在弦上产生 (n + 1) 个波节和 n 个波腹。

- 这些波腹的位置是严格确定的。对于第 n 次谐波，其波腹位于距离琴弦一端（如琴桥）(2k-1) * L / (2n) 的位置，其中 k = 1, 2, ..., n。也就是说，第 n 次谐波的 n 个波腹，会均匀地分布在整个弦长上。

- 由此可得一个重要结论：基波 (n=1) 只有一个波腹，它位于弦的正中央。而**琴颈拾音器**通常安装在靠近弦中央的区域（Les Paul 一般是靠近琴桥的琴弦四等分点），因此它天然地靠近基波和低次谐波（如2次、3次谐波）的波腹，能强烈地采集到这些决定音色主体和厚度的频率成分。

- 相反，高次谐波 (n 较大) 的波腹数量多且分布密集。**琴桥拾音器**所在的位置，虽然远离基波的波腹（Les Paul 一般是靠近琴桥的琴弦十六等分点），却很可能与某个高次谐波（例如第8次谐波）的波腹位置重合。这使其能优先采集到那些赋予音色明亮度、冲击力和“金属感”的高频成分。

### 问：单线圈拾音器和双线圈拾音器有什么区别？

- **单线圈拾音器**的特性是，它容易受到环境中电磁场（如灯光、电脑）的干扰从而采集到噪音。**双线圈拾音器**通过**两个线圈反向缠绕**，可以有效抵消这种噪音。如果噪音过大，检查一下是否面对干扰源，或者尝试在房间的不同位置练习。

### 问：综合效果器音箱是什么意思？

- 你知道的，电吉他的信号需要处理才能形成音色，信号的处理流程就是**效果链**。

- 处理信号一般需要多种变换，实现这一变换的设备就叫做**效果器**。

- 如果一个效果器只能实现一种变换，则称为**单块效果器**；如果一个效果器能实现多种变换（一般是电子的），则称为**综合效果器**。

- 在传统的由单块效果器组成的效果链中，串联顺序一般是这样的：

    `吉他 -> 调音表 -> 噪音门 -> 压缩器 -> 哇音踏板 -> 均衡 -> 增益 -> [调制 -> 延迟 -> 混响] -> 放大器 -> 音箱箱体`

- 现在我们来介绍**箱头**，其包含了**前级**和**后级**两部分，箱头的功能是塑造信号的音色并放大功率以驱动音箱。

- `均衡`和`增益`部分称为箱头的**前级**，用于塑造音色并初步放大至足以驱动后级。它负责处理来自吉他的微弱信号，通过调节增益（GAIN）来**决定音色的失真度和饱和度**（从清澈到 crunch 再到高增益失真）；通过均衡（EQ，通常是三段：低频、中频、高频）来**塑造音色的骨架**（比如增强低频让声音更厚实，提升高频让声音更明亮）。

- `放大器`部分称为箱头的**后级**，用于放大信号功率并改变信号的阻抗匹配特性。它接收来自前级或效果环路的信号，并将已经塑形完毕的**低功率、高阻抗信号**转换为**高功率、低阻抗的信号**，从而拥有足够的能量去驱动扬声器（喇叭）这种低阻抗负载。

- 箱头的前级和后级之间设有一个**效果环路（FX Loop）**，它是一个由 SEND 和 RETURN 两个接口组成的回路。乐手可以将`调制`、`延迟`、`混响`等效果器连接在这个回路中，使这些空间类效果工作在前级失真之后、后级放大之前，以获得更清晰的尾音。它们为干瘪的原始信号添加空间感和趣味性。混响**模拟在不同空间中的反射**，制造深度；延迟**制造回声或增加延音**，增加层次感；调制效果则**让声音流动、旋转起来**。这些效果器是可以选择安装与否的，如果选择不安装，前级的输出信号会直接输入给后级。

- 市面上大部分数字综合效果器都可以模拟`调音表 -> 噪音门 -> 压缩器 -> 哇音踏板 -> 均衡 -> 增益 -> 调制 -> 延迟 -> 混响`这一部分（具体构成和顺序可能不同）。再进一步，如果把数字综合效果器和`放大器`、`扬声器`集成在同一外壳内，就得到了通常所说的**综合效果器音箱**（更准确的术语是“数字建模一体式音箱”）。它是一套完整的“一体化解决方案”，无需外接任何设备即可获得多种可编程音色。

### 问：我的琴桥怎么是斜的？

- 不用担心，这不是设计或制造瑕疵，而是**补偿式琴桥**设计。

- 这一设计首先补偿了**低音弦的弦高**，也就是到指板的距离。低音弦更粗且振动幅度更大，需要更大的弦高以防止打品，于是补偿式琴桥的高度随琴弦振动频率降低而逐渐升高。

- 其次它补偿了**低音弦的频率**。当手指按下低音弦时，琴弦的横向位移远大于同品位高音弦。为了在按弦时避免更大的横向位移造成音高偏移，把琴弦加长可以让“稀释”这一位移以稳定音准，于是补偿式琴桥的位置随琴弦振动频率降低而逐渐远离弦枕。
    > `根据白金汉 π 定理，任何一个由有量纲的物理量参与的物理过程中的函数关系都可以转换成某些无量纲量之间的函数关系。琴弦横向位移与有效长度的比 x/L 是一个非常重要的无量纲量，其在研究琴弦横向变形对振动特性的影响时记载了琴弦变形的几何特征。因为低音弦比较高，从而 x 更大，自然就要把 L 做大一点以保证不同琴弦的变形特征一致。`

### 问：摇把是什么？怎么用？

- 我们知道，琴桥的位置决定了琴弦的有效振动长度，从而与琴弦张力共同决定振动频率。如果让琴桥可以绕支点旋转，并连接一根「摇把」来控制旋转角度，即可动态改变琴弦的张力和有效长度，从而连续调节音高——这一机械装置称为**颤音系统**（tremolo system）。摇动摇把即可产生音高起伏的效果。

- 根据摇把允许的运动方向及琴桥的机械结构，市面上的颤音系统可分为以下三类：

1. **单摇（Down-only Tremolo）**  
   琴桥底面紧贴琴体，摇把只可下压。下压时琴弦松弛，音高下降；释放后琴桥由弹簧拉回原位，音高复原。**无法上拉（不能升高音高）**。典型代表：Fender 同步颤音系统（Synchro Tremolo）在标准设置下。

2. **双摇（Double-locking Tremolo）**  
   俗称「大双摇」。典型结构为 Floyd Rose 或类似设计：  
   - 琴桥为**浮动式**，摇把既可下压也可上拉，音高可双向变化。  
   - 琴弦在弦枕处和琴桥处均被**机械锁定（弦锁）**，配合琴桥后端的**微调旋钮**，用于锁定后在有限范围内校准音高。  
   该设计牺牲了调音便利性，换来了极大的音高浮动范围和优异的音准稳定性。

3. **小双摇（Floating Tremolo / "Small" Double-locking）**  
   这并非独立的硬件类别，而是**将传统单摇系统通过调节弹簧平衡设置为「悬浮状态」**：琴桥尾部微微翘起，脱离琴体表面，从而获得有限的上拉能力（通常不超过全音）。  
   - 无弦锁，调音换弦比大双摇方便。  
   - 上拉幅度小，主要用于颤音或小幅升调，无法实现 Floyd Rose 式的大幅度俯冲/拉升。  
   - 由于未改变硬件结构，单摇与小双摇之间可**完全可逆转换**（仅需调整弹簧拉力与琴桥高度）。
    > `乐手群体习惯上常将「大双摇」等同于双锁式颤音系统，将「小双摇」等同于悬浮设置的单摇系统。严格学术上，只有具备弦枕+琴桥双锁定机构的才应称为「双锁颤音系统」（double-locking tremolo），而「单摇」「小双摇」均属于非锁定式浮动琴桥的不同工作状态。但行业习惯已广泛接受「大/小双摇」这对简称，本指南在此框架下行文。`

- 值得注意的是，颤音系统的调节（尤其是弹簧张力平衡、音准稳定性维护）远比固定琴桥复杂，双锁式大双摇尤甚。如果主要练习节奏、Riff 或推弦技巧，固定琴桥或单摇（非悬浮设置）是更省心的选择。若对摇把技巧感兴趣，可从单摇或小双摇起步——它们提供了基础的摇把体验，又避开了大双摇繁琐的维护成本。

## 🎛️ 彩蛋：声卡软效、单块管箱怎么玩？ {#advanced}

- 这一部分是写给那些已经不满足于“能响”，想要探索更多音色可能性的乐手。我们会依次聊两套系统：**声卡 + 软效**和**单块 + 管箱**。它们分别代表了两种不同的进阶路径：数字模拟和真实味道。未来可能会更新 **DAW** 和**录音声卡**的玩法，以下提到的声卡都默认是入门声卡，也就是只能执行基本功能，沟通吉他和电脑的声卡。

1. **声卡 + 软效：年轻人的第一台万能箱头**

- **声卡**简单来说是在电吉他模拟信号和电脑听得懂的数字信号间进行 AD/DA 转换的设备。现在的进阶级声卡还集成了话放（麦克风放大）、耳放（耳机放大）和48V幻象电源。对于吉他手来说，最关键的是它提供了高阻抗输入（Hi‑Z，通常 1MΩ 左右），可以直接怼进电吉他而不损失高频——这点是普通电脑板载声卡做不到的。

- **软件效果器**则是一类运行在电脑或手机上的数字信号处理程序，用软件算法模拟传统效果链中的所有环节——压缩、过载、均衡、调制、延迟、混响，甚至完整的箱头 + 箱体模拟。软件效果器作为运行在电脑上的程序，接收来自声卡的数字信号，通过数字信号处理（DSP，Digital Signal Processing）算法来添加音色并将处理过的数字信号传回声卡。这里我们发现，电脑只能处理数字信号，而电吉他的信号永远是模拟的（不要跟我说什么自动挡吉他，那是玩具）。这就意味着声卡必须充当“翻译官”，执行 AD/DA 转换，让你的吉他、电脑、音箱（或耳机）能够沟通。

- 你可以在屏幕上像拼乐高一样拖动虚拟单块，调节旋钮，并且支持保存/切换上百种预设。知名软效包括：[Amplitube](https://cn.ikmultimedia.com/products/amplitube5/)、[Guitar Rig](https://www.native-instruments.com/zh/products/komplete/guitar/guitar-rig-7-player/)、[Neural DSP](https://neuraldsp.com/) 系列、[Bias FX（移动端）](https://www.positivegrid.com.cn/bias-fx-mobile)、[Bias FX2（电脑端）](https://www.positivegrid.com.cn/bias-fx2)、手机端的 [Tonebridge](https://www.mu.se/tonebridge)。软效的本质是用 CPU/DSP 芯片的算力，实时计算出效果器对信号的变换结果。性能开销方面，只要你的手机电脑不是砖头，延迟都不明显。

- 具体来说，买一块入门级声卡（比如 [Focusrite Scarlett Solo](https://item.jd.com/10097990517420.html)、[Audient EVO 4](https://item.jd.com/10045860639176.html)，或者更便宜的 [Behringer UM2](https://item.jd.com/52558428792.html)、[Hotone JOGG](https://item.jd.com/68175243382.html)，最好有独立驱动），然后在电脑或手机上安装对应的声卡驱动和软件效果器（如 [Amplitube](https://cn.ikmultimedia.com/products/amplitube5/)、[Guitar Rig](https://www.native-instruments.com/zh/products/komplete/guitar/guitar-rig-7-player/)、[Neural DSP](https://neuraldsp.com/) 免费试用版，甚至手机上的 [Tonebridge](https://www.mu.se/tonebridge)）。把吉他接到声卡，声卡接电脑或手机，耳机或音箱插在声卡的输出口，打开软件，选好驱动和设备，调整一下 ASIO 参数（缓冲长度不宜过高，128 或 256 即可，若对设备性能十分自信可选 512），你就能获得成千上万种音色，还能直接录音。

- 这个方案的好处是：
    - **音色上限高**：顶级软件效果器的音色已经能和真实箱头叫板。如果你愿意在 [JAM BUDDY II](https://item.jd.com/10174949443805.html) 的基础上加一块入门声卡和免费软效，你可以把综效和软效结合起来搞出更炸裂的音色。
    - **扩展性强**：你随时可以加鼓机、录音、混音，甚至玩一玩 MIDI 键盘。
    - **不扰民**：全在耳机里，半夜撸琴也不怕邻居敲门。
    - **可以玩 Rocksmith 2014 Edition - Remastered**：只需要配合 Github 上免费的程序 [RS_ASIO](https://github.com/mdias/rs_asio) 就可以避免购买昂贵而用处少的育碧原厂 Real Tone Cable 音频线。不瞒大家说，我刚开始学民谣吉他的时候就想过要玩 Rocksmith 这款游戏，结果发现需要用电吉他玩。我试过给木吉他装拾音器，买了电吉他之后还买过移动声卡、淘宝上不知哪来的“兼容 Rocksmith 音频线”，总之就是想过要玩 Rocksmith 还要接软效。踩了这么多坑之后我梳理明白了，新手最全面的配置无非是一个 JAM BUDDY II 再加上一块 HOTONE JOGG 声卡，以及 Github 上最具互联网精神的项目。我为什么要指名道姓说是 Rocksmith 2014 Edition - Remastered 呢？因为别的版本我没玩过。虽然重制版相比原版不能走正规途径添加 DLC 歌曲，但是关注一下[奔奔史密斯](https://space.bilibili.com/293607656)老师就知道该怎么添加了。

- 坏处呢？
    - **需要电脑**：练琴要开电脑、开声卡、开软件，懒癌患者容易劝退。
    - **有一定的延迟风险**：虽然现代电脑配合 ASIO 驱动能做到几乎无感，但没设置好时，那几十毫秒的延迟会让你感觉在弹空气。
    - **没有“怼音箱”的爽感**：软件效果器进监听耳机或者监听音箱，声音太“干净”，缺少物理箱体那种轰胸口的震动。有些老炮就是受不了这个。

2. **单块 + 管箱：玩块穷三代，玩管毁一生**

- **单块效果器**，简称“单块”，是一种只处理一种声音效果的小盒子。比如：过载、失真、延迟、混响、合唱、压缩……每一个单块只做一件事，但把它做得尽量纯粹。我记得我之前讲过了，在效果链那里，但我不介意再说一遍。你可以像拼乐高一样，用音频线把它们按顺序串起来，组成你自己的实体效果链。“乐高”的比喻我也用过一次了，上次用说的是软效的效果链组合。每个单块通常有一个脚踏开关，方便你在演奏时随时踩开或关闭。单块的魅力在于：它们的电路往往是模拟的（不像数字综合效果器那样把声音采样、运算后再输出），声音更具个性，动态更自然。很多老炮觉得单块比综合效果器更有“味道”。但是，单块的世界也是一个深坑——你需要专门的电源（而且电源可能带来噪音）、需要很多短的连接线（patch cable）、需要一个效果器板来固定它们，还要研究串并联顺序和信号增益结构。调一套好用的单块板子，比在软件里拖图标麻烦得多。

- 当你开始给多个单块供电时，通常会遇到两个方案：便宜的“菊花链”和看着更专业的“大砖头”多路电源。很多人都想当然地认为大砖头电源是升级，但现实往往更复杂。

### 🔗 菊花链：简陋但有时行之有效

- 这是每个吉他手大概率都接触过的方案。原理就是一个电源适配器，通过一根分线，把几个插头串联起来，同时给几个效果器供电，成本很低。

- 这小破烂听起来不错，但有个致命缺陷：所有效果器的地线都是连在一起的，会形成**地线回路**（Ground Loop）。不同效果器之间微小的电位差，会在这个回路里产生循环电流，这些电流混杂着各种噪音，就会从你的音箱里播放出来。
    > `地线回路（又称地环路）的解释放在下文更合适。`

- 这小玩意混用的弊端尤其明显：数字效果器与模拟效果器混用时，数字电路里的高频时钟噪声会通过共用的地线，污染旁边的模拟效果器，产生刺耳的噪音。一些对电流需求大的效果器，通过菊花链供电也可能无法正常工作。

### 🧱 大砖头（多路非隔离电源）：可能是升级，也可能是换了个包装的坑

- 看着更专业的“大砖头”电源（如早期版本的Vitoos、Mosky、Joyo JP-02等），它们大多属于“非隔离”电源。这意味着它内部可能只是把大电流分成几路输出，各个输出口的电路最终仍然共用一个“地”，本质上和菊花链是一样的。所以，接上以后，地线回路的隐患依然存在，该来的噪音还是可能会来。
    > `关于JOYO JP-02，JOYO官网曾明确指出，JP-02 **不是“真隔离”** （true isolation）电源。这能解释为什么有人觉得它比菊花链好，也有人觉得换了个寂寞。`

### 🔌 “真旗舰”：隔离电源——为什么它是进阶玩家的选择

- 真正的“一劳永逸”是**隔离电源**。它为每个输出口都配备了独立的电路，确保每条供电路径在电气上都是完全独立的，从物理上切断噪声。这东西能消除地线回路，防止数字与模拟电路间的串扰。代表产品有 Voodoo Lab Pedal Power、Strymon Zuma、Cioks DC7等。

- 不过，即便隔离电源性能达峰，对一些高增益变态玩家，可能仍有底噪。这时，可能需要考虑在效果链中单独加一块高品质的降噪单块（Noise Gate）来“关门打狗”。

- 选择电源，本质上就是在“**预算**”和“**纯净度**”之间做权衡。希望这份“避坑”指南，能帮你少花冤枉钱。

- **电子管音箱**，简称“管箱”，指的是放大电路使用**真空电子管**（而不是晶体管）的音箱。管箱是电吉他声音的“终极归宿”之一。为什么？因为电子管在过载时会产生一种**平滑、温暖、动态极其丰富**的失真——你轻弹，就是清音；你用力弹，就自然过载。它对手指的力度、拨片的角度、音量旋钮的细微变化都非常敏感，几乎是“把你的弹奏习惯完全暴露出来”。无数经典摇滚、布鲁斯、金属的录音室声音都来自管箱。但管箱也有很多“反人性”的缺点：贵、重、电子管会老化（需要更换）、音量巨大（早期100瓦管箱在家里根本没法用）。好在现在很多小功率管箱（比如 [Marshall DSL1CR](https://www.marshall.cn/product/MD00327?propId=286) ）带功率切换功能，可以在 1W 甚至 0.1W 的功率下得到好听的管味，同时不打扰邻居（注意晶体管音箱一般指全机功率，而电子管音箱的功率一般指电子管功率）。一句话：一台好的管箱，再加上你的吉他和几块单块，基本就是吉他手的“毕业套装”了。

- 当你把单块和管箱串联在一起，你可能会遇到下面这个吉他手最经典的鬼故事：我搭好效果链了，但是底噪怎么这么大？我一碰琴弦就小了又是怎么回事？

- 这是一个非常经典的问题，根源往往出在**效果器的电源适配器**上。但是，我们要先搞清楚你听到的底噪是哪种特征，主打一个对症下药：
    - **50Hz/100Hz 的低频噪声**：这是**市电工频及其谐波**。它来自电源适配器内部的**变压器寄生电容**——无论开关电源还是线性电源，只要变压器初次级之间存在分布电容，220V 工频电压就会通过这个电容泄漏到效果器的信号地，使效果器的金属外壳、音频线屏蔽层对真实大地带上一个**几十伏的工频交流电压**。这个电压虽然电流极小（微安级），不会让人触电，但重要的是，这一电压通过两个途径向大地泄放：一个是音频线地端与真实大地（或人体）之间存在的寄生电容允许了交流电通过而不会经过拾音器；另一个则是音频线地端**经过拾音器线圈**到达音频线热端，再由热端导体与真实大地（或人体）之间存在的寄生电容流向大地。
        > `任何挨得不太远的导体都可以构成电容器。为什么要强调是否经过拾音器呢？你无妨验证一下，吉他上的旋钮会对噪音产生影响，所以只有经过了拾音器的那部分信号才是我们能听到的噪音。我还验证过，如果用电池给单块供电接到音箱（电池供电）上，这个噪声就会消失。拾音器线圈分到了信号地的交流电压才是底噪的根本原因，铁证如山。至于两脚插头、三脚插头供电的音箱的情况，如下文所述。`
    - **几千赫兹到几十千赫兹的高频噪声**：这是**开关电源特有的高频开关噪声**。开关电源为了做小做轻，内部以几十 kHz 到几百 kHz 高速开关，这个开关方波造成的输出电压波动（称为“**纹波**”）也会通过拾音器线圈和寄生电容泄漏到输出地，携带大量高次谐波。频率高，更容易通过拾音器线圈转换成可听噪声，听感比工频更刺耳。
        > `根据典型的频率响应模型，感抗模正比于交流电频率，而容抗模则是反比于交流电频率。而拾音器线圈和寄生电容是串联关系，阻抗大则分压大。`

- 而当手指触摸琴桥、琴弦、效果器踩钉等金属部件（即吉他电路的地线）时，人体对大地近似构成直通。这相当于给信号地的电压提供了一个低阻抗的对地泄放路径，从而减少噪音。

- 解决方法很多，丰俭由人：
    - **护弦**：护弦让信号地-大地电容的两端互相接触从而短路这个电容。
    - **换线性电源供电**：常见的线性电源都是可以同时为很多单块效果器提供线性直流电的（不通过高频通断来调节功率，而是直接给出需要的电流）。没有了高频纹波自然就没有这个困扰了。
    - **换隔离电源供电**：极致的破财消灾，不管是花费还是效果。隔离电源指的是将变压器初次级线圈尽可能隔离开，大幅度减小寄生电容（这才是把市电带到琴桥的罪魁祸首）。
    - **换干电池供电**：化学电池的电流肯定不是交流电。打死不要图省事用可充电电池，可充电电池内部电路是新的不稳定因素。
    - **给吉他做屏蔽**：就是在拾音器周围贴上金属箔用以实现电磁屏蔽。当然，当噪声的周期小于导体内电子达到静电平衡的弛豫时间时，静电屏蔽会失效。当然，在吉他和市电的领域，我们只能接触到工频（50Hz）和射频（kHz~MHz），这点频率还是不足以让静电屏蔽失效的。
    - **换一台音箱**：换音箱解决问题不是玄学。你可能会问：我明明用的是电池效果器，音箱也是好的，为什么换一台音箱底噪就小了？答案在于音箱的电源接地类型。两脚插头（浮地）音箱：无论音箱内部是线性电源还是开关电源，它的信号地都会通过变压器寄生电容耦合到零线，带上一个几十伏的工频共模电压。这个电压通过音频线传到吉他地，再经过拾音器转换成底噪。即使效果器用电池，也救不了——因为噪声源在音箱这边。手摸琴桥能临时泄放，但松手就回来。三脚插头（接地）音箱：电源地线直接连接真实大地，音箱的信号地被强制拉到 0V 。只要效果器也用电池（或干净电源），整个系统就实现了单点接地，没有地环路，也没有悬浮的共模电压，底噪只剩下环境磁场感应和元件热噪声——通常小到可以忽略。这里说一个反直觉的亲身经历：笔者有朋友用两脚插头音箱 + 电池效果器，底噪巨大；换成两脚插头音箱 + 两脚电源效果器，底噪反而小了。这是因为两个两脚电源的寄生电容并联，给噪声提供了更低阻抗的泄放路径，实属歪打正着。但这不是可靠设计，换房间、换适配器都可能打回原形。所以，“换一台音箱”的正确姿势是：换成三脚插头、且墙插接地良好的音箱。 不需要贵，哪怕是一台二手的三脚小晶体管箱，只要接地可靠，配合电池效果器，就能让你告别“手摸琴桥”的临时抱佛脚。如果你已经有了一台心爱的两脚管箱，也可以考虑请专业人士加装接地线（但有一定风险，不推荐新手DIY）。
        > `地环路指的是一个系统多点接地，导致大地导通了这些接地点所形成的回路，这个回路将系统噪声送回原系统内部消化，甚至还会把其他系统的噪声送进来。所以要么做好单点接地（例如只有音箱用三脚插头），要么在浮地系统中避免引入工频噪声（隔离电源/干电池）和纹波噪声（线性电源/干电池）。`

- 好了，把电源引入的噪声解决了。但怎么还有噪声？别急，这些就是不可避免的环境噪声了，例如：
    - **工频磁场直接感应**：你的拾音器线圈就是一个天线。家里墙里走的电线、楼下的大变压器、桌面的音箱电源线、甚至隔壁房间的空调压缩机，都会向外辐射50Hz的磁场。这个磁场直接穿过拾音器线圈，根据法拉第定律产生感应电动势——你听到的就是那种沉稳的“嗡嗡”声，手摸琴桥也消不掉（因为这不是电压泄漏，是磁场穿心）。双线圈拾音器能抵消一部分，但做不到绝对零。想要解决的话就尝试旋转身体或移动位置，找到一个磁场最弱的方向（通常在房间角落或与墙壁平行）；用双线圈拾音器；把音箱电源线远离吉他。
    - **电阻热噪声（约翰逊噪声）**：任何有电阻的元件（拾音器线圈、电位器、电阻）在绝对零度以上都会产生微弱的随机电压，频谱覆盖全频段，听起来像“嘶——”的背景沙沙声。增益越高越明显。这是物理极限，没办法彻底消除。只要你不把增益和音量开到非人类水平就行了，你那 1145141919810dB 增益的音色没人想听。如果必须高增益，加一块噪声门（比如ISP Decimator）在效果链中，不弹的时候自动静音。
    - **环境高频电磁干扰**：手机信号、Wi-Fi、蓝牙、节能灯、电脑屏幕……这些kHz到GHz的辐射也可能被拾音器或效果器电路检波，产生杂音。通常表现为不规则的“吱吱啦啦”或“嘀嘀”声。尝试把手机放远一点，或者用质量好的屏蔽线。实在不行给吉他电路仓做铜箔屏蔽（对电场有效，对磁场无效，但聊胜于无）。
